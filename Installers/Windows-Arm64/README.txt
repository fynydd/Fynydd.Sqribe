To install SQribe follow these steps:

1. Open a PowerShell window as Administrator (right click PowerShell and choose run as adminiistrator)
2. Navigate to the folder where this README file is
3. Run the following command:

Set-ExecutionPolicy Bypass -Scope Process -Force; ./install.ps1

You will be asked for your password so the install script can copy files and set up your PATH.
Once complete, quit the PowerShell and open a new one (or a command prompt) to begin using SQribe.
